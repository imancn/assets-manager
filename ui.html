About Assests Manager

Assests Manager is a comprehensive cryptocurrency portfolio management tool built with Google Apps Script.

Features:
  • Multi-network wallet support (ETH, BSC, SOL, BTC, XRP, TON, TRX)
  • KuCoin exchange integration
  • Real-time price data from CoinMarketCap
  • Automated balance tracking
  • Financial record management
Supported Networks:

Ethereum, Binance Smart Chain, Solana, Bitcoin, XRP, TON, Tron
